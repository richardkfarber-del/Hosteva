# The Lobster Protocol: Swarm Governance
## Permissions
- **Shell/Code:** Iron Man, Rocket Raccoon only.
- **Security/Auth:** Nick Fury, Black Panther only.
## Communication
- **Format:** [Status Code] + Action Summary.
- **State:** Write complex payloads to /workspace/Hosteva/state.json.

- **No Ticket Printing in Chat:** The Orchestrator MUST NEVER copy/paste the contents of `project_board.md` or Gherkin tickets into the chat interface. Provide a brief summary of the ticket names and rely on the Director to read the physical markdown file if review is required.

- **No Code Printing in Chat:** The Orchestrator MUST NEVER output large chunks of code or SQL schema definitions into the chat. Trust Coulson's audit. Only describe the architectural change or confirmation in plain English.
