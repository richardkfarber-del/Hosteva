# GO-TO-MARKET STRATEGY (V1.0)
**Strategist:** Hawkeye (AGENT-03-PRODUCT)
**Scope:** State of Florida (Statewide Initial Launch)

## 1. The Core Funnel
The primary vector for user acquisition and immediate revenue generation is the **Eligibility-to-Posting Pipeline**:

1. **The Hook (Free): Address Eligibility Check**
   - User inputs a Florida property address.
   - Hosteva cross-references state, county, and municipal zoning laws.
   - **Branch A (Ineligible):** User is informed the property cannot be used for STRs. (Saves user time/money; builds extreme brand trust).
   - **Branch B (Eligible):** User receives a "Green Light" and a high-level summary of required permits.

2. **The Conversion (Paid): Subscription Trigger**
   - To unlock the actual permit applications, document generation, and the "Post to Airbnb/Vrbo" automation, the user must subscribe to Hosteva's premium tier.

3. **The Value Delivery:**
   - Hosteva auto-fills state DBPR forms and local tax registrations.
   - Hosteva generates search-safe, compliance-verified listing descriptions.

## 2. Updated Backlog Focus (Phase 1 Crucible)
To execute this GTM, the immediate Sprint Backlog is being restructured to prioritize the conversion gate and the document generation engine, as the Address Eligibility module is now complete.
