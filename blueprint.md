# Hosteva Technical Blueprint v1.0

## System Architecture
- **Frontend:** React 19 (Mobile-first, PWA-ready)
- **Backend:** Node.js 22 (Fastify)
- **Database:** PostgreSQL (Supabase/Local)
- **AI Integration:** OpenClaw 2026.3.23 Swarm

## Core Modules
1. **Compliance Engine:** Real-time monitoring of FL municipal codes.
2. **Listing Optimizer:** Dynamic sync between Airbnb/VRBO via official APIs.
3. **Host Dashboard:** A unified view of "Eligibility Status" and "Listing Health."

## Tech Stack Requirements
- Agents must use **ESM (ECMAScript Modules)**.
- All UI components must follow the **Stitch-Design** (Teal/White/Slate) token system.
- Deployment target: Render (SlingShot Ring).
